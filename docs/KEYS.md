## Common Remapping Examples

### Swap CapsLock and Escape (Vim users)

```rhai
fn on_init() {
    remap("CapsLock", "Escape");
    remap("Escape", "CapsLock");
}
```

### CapsLock as Control

```rhai
fn on_init() {
    remap("Caps", "Ctrl");
}
```

### Disable CapsLock entirely

```rhai
fn on_init() {
    block("CapsLock");
}
```

### Arrow keys on home row (with modifier)

```rhai
// Use a layer or chord for this in practice
fn on_init() {
    // H/J/K/L to arrows
    remap("H", "Left");
    remap("J", "Down");
    remap("K", "Up");
    remap("L", "Right");
}
```

### Numpad navigation when NumLock is off

```rhai
fn on_init() {
    // Already default behavior, but explicit:
    pass("Numpad4");  // Left on numpad
    pass("Numpad6");  // Right on numpad
    pass("Numpad8");  // Up on numpad
    pass("Numpad2");  // Down on numpad
}
```

### Media keys remapping

```rhai
fn on_init() {
    // Remap F7/F8/F9 to media controls
    remap("F7", "MediaPrev");
    remap("F8", "MediaPlayPause");
    remap("F9", "MediaNext");
}
```

## Notes

- **Case insensitivity**: `CapsLock`, `capslock`, `CAPSLOCK`, and `Caps` all refer to the same key.
- **Alias preference**: Use the canonical name (first column) for clarity, but aliases work identically.
- **Unknown keys**: Keys not listed here will produce an error. Check spelling and use the exact names shown.
- **Platform differences**: Some keys may not be available on all platforms (e.g., `AltGr` on US keyboards).
