### Integrations

#### Integration
- **Description:** PanicGuard automatically records panics and recoveries in telemetry when executing protected code
- **Frontend Component:** PanicGuard::execute
- **Backend Endpoint:** record_panic / record_recovery
- **Data Flow:** Protected code executes → Panic caught → record_panic() → Telemetry updated → Success → record_recovery()

#### Integration
- **Description:** CircuitBreaker records open/close events in telemetry when state changes
- **Frontend Component:** CircuitBreaker::open_circuit / close_circuit
- **Backend Endpoint:** record_circuit_breaker_open / record_circuit_breaker_close
- **Data Flow:** Failure threshold exceeded → open_circuit() → record_circuit_breaker_open() → Recovery succeeds → close_circuit() → record_circuit_breaker_close()

#### Integration
- **Description:** Flutter UI polls telemetry via FFI to display recovery notifications
- **Frontend Component:** Flutter UI (periodic polling)
- **Backend Endpoint:** keyrx_telemetry_snapshot_json
- **Data Flow:** Flutter timer → FFI call keyrx_telemetry_snapshot_json() → Parse JSON → Display notifications for panics/recoveries
