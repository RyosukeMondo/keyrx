### Integrations

#### Integration
- **Description:** Rhai layer_* functions queue validated PendingOps, update an in-script preview LayerStack, and sync into RemapRegistry’s LayerStack during apply_pending_ops for consistent layer control
- **Frontend Component:** Rhai runtime scripts
- **Backend Endpoint:** RemapRegistry/LayerStack
- **Data Flow:** Script calls layer_* -> pending ops + preview updates -> apply_pending_ops resolves actions to layer ids -> registry LayerStack holds definitions and mappings
